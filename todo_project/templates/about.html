Este projeto é uma aplicação web de gerenciamento de tarefas desenvolvida com Flask.

O objetivo é oferecer uma maneira simples e eficiente para que os usuários possam organizar seu dia a dia, criando, visualizando, editando e excluindo tarefas.

Funcionalidades

  • Autenticação de usuários (registro, login e logout)
  • Criação de tarefas
  • Visualização de todas as tarefas cadastradas
  • Atualização e exclusão de tarefas
  • Configurações de conta (alterar nome de usuário e senha)
  • Proteção de rotas para usuários autenticados

O que ele possui

  • Página de registro/login: acesso seguro à aplicação
  • Menu lateral: links para adicionar e visualizar tarefas
  • Página de tarefas: lista de tarefas com opção de editar ou excluir
  • Menu de conta: permite alterar usuário/senha ou sair
